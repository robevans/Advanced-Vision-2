Run RGB histogram comparison example as:

compare

- in the matlab command window. You will be prompted for two image locations and the resulting 
RGB histograms and comparison measures will be displayed. A set of sample images as used in 
lectures is provided in the images/ diretcory.

e.g. 

>> compare
Enter first image file name
?images/monster.jpg
Enter second image file name
?images/monster1.jpg

bhattacharyyaDistance =

    0.2942


chiSquaredDistance =

    0.2866


EuclideanDistance =

    0.0295
>>
